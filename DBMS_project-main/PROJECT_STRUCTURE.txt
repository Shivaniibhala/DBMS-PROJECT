==========================================
KEY FILES TO EDIT:
==========================================

1. backend/.env (create from .env.example)
   - Set database credentials
   - Set JWT secret
   - Configure email settings

2. backend/config/schema.sql
   - Run this to create database structure

3. To load sample data:
   - backend/utils/loadData.js
   - backend/utils/loadCSV.js

==========================================
INSTALLATION STEPS:
==========================================

1. Setup Database:
   mysql -u root -p < backend/config/schema.sql

2. Install Backend:
   cd backend
   npm install
   # Create and configure .env
   npm start

3. Install Frontend:
   cd frontend
   npm install
   npm start

4. Access App:
   http://localhost:3000 (Frontend)
   http://localhost:5000 (Backend API)

==========================================
API ENDPOINTS:
==========================================

Authentication:
  POST   /api/auth/signup
  POST   /api/auth/login

Scholarships:
  GET    /api/scholarships          # List all (with filters)
  GET    /api/scholarships/:id       # Get one

Wishlist (Protected):
  POST   /api/wishlist               # Add to wishlist
  GET    /api/wishlist               # Get user wishlist
  DELETE /api/wishlist/:id           # Remove from wishlist

==========================================
DATABASE TABLES:
==========================================

1. users           - User accounts
2. scholarships    - Scholarship data
3. wishlist        - User favorites
4. email_alerts    - Alert tracking
